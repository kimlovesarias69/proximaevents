# Eros Question Generator

A simple tool for exploring Eros through conversation at Proxima.

Built as a lightweight webpage for salon events, this generator offers randomly selected prompts designed to invite reflection on desire, longing, and aliveness—both in yourself and in those around you.

## Usage

Open the page and tap “Give me a question” to generate a prompt.

Use it as a starting point for conversation or personal reflection.

## Structure

- `index.html` — main interface
- `questions.json` — question library

## Notes

Designed to be fast, minimal, and mobile-friendly for live event use.
